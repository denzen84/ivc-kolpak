## Overview
IVC KOLPAK is smart CCTV system that alarms into Telegram if motion detected in the camera area. It provides intelligent video control of the object, because only motion video clips (as 'gif' animated media) and pictures of motion object passes into Telegram chat, group or channel. This gives features to reduce human attention for CCTV system and provides Internet traffic economy. It comprises the principles of simplicity and convenience, and low cost of ownership. It's best for far objects where only available 3G/4G Internet connection.
